Project for Git exercises



Pytanie 1
Git wykonuje Fast-forward, gdy gałąź główna nie ma własnych commitów od momentu rozgałęzienia.
Git wykonuje Merge commit, gdy gałąź główna miała własne commit-y, tworzy się nowy commit łączący historię.

Pytanie 2
Merge zachowuje rozgałęzioną historię, powstaje merge commit.
Rebase przenosi commity na koniec głównej gałęzi, tworząc liniową historię.
